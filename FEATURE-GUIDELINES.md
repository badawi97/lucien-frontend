# Feature Development Guidelines

## Every Feature Must Have
- Models
- DTOs
- Service
- Routes
- Components
- Permissions
- Localization
- Tests

## Recommended Structure
features/
  users/
    pages/
    components/
    services/
    models/

## Feature Checklist
- Follows architecture
- Uses existing patterns
- Permission protected
- Localized
- Tested
- Responsive UI
