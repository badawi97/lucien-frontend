# Lucien Coding Standards

## TypeScript
- Enable strict mode.
- Avoid any.
- Use interfaces and DTOs.

## Naming

Components:
- UserListComponent
- UserEditComponent

Services:
- UserService
- PermissionService

DTOs:
- UserDto
- CreateUserDto
- UpdateUserDto

## Angular Rules
- Standalone components.
- Reactive forms.
- Lazy loading.
- OnPush where appropriate.

## Forbidden
- Business logic in components.
- Direct HttpClient calls from components.
- Hardcoded URLs.
- Console logs in production.

## Code Review Checklist
- Readable
- Reusable
- Secure
- Performant
- Testable
